# Bitácora de Desarrollo - Tarea QuickSort

Este archivo cumple con el requerimiento de entregar el texto utilizado para generar el código mediante Inteligencia Artificial.

## Prompt utilizado en Gemini:
"me puedes ayudar con esto y desirme el paso a paso de como ir haciendolo gracias Desarrollar la implementación del algoritmo QuickSort presentado, en Java, para su entrega deberá consignar el archivo o proyecto Java, con un archivo en pdf, que muestre una gráfica de varios escenarios que haya probado para su solución. Entrega 2 archivos en un repositorio de github, uno extensión .java y la gráfica en .pdf y un archivo .MD con el texto que usaron con ChatGPT para generar el código."

## Detalles de la Implementación:
- **Lenguaje:** Java
- **Algoritmo:** QuickSort (Divide y vencerás)
- **Escenarios probados:** Aleatorio, Ya ordenado y Orden inverso.
- **Herramienta IA:** Google Gemini
